Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

There are a number of ways to secure a website from web security attacks. Some of these ways are to make sure that:

  • Ensure that all data is sanitized before it is accepted by the system, either manually or through trusted third-party solutions
  • Ensure that the website uses a secure authentication and open-standard authorization protocol or framework. Make use of well-known and trusted third-party applications, when possible.
  • Ensure that the website is built using a continuous integration/continuous deployment workflow. This can catch oversights made by developers when changes occur in the codebase, and reject changes before reaching production.
  • Ensure that the network the website is hosted on is secure.
  • Automaticaly timeout inactive users to protect user credentials
